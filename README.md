# Contributing

First install `esy` (use 0.4.3 or later for correct lock format)

```
npm install -g esy@0.4.3
```

Install (once, and when package.json changes)

```
esy install
```

Building / Testing

```
esy build && esy test
```

Or use watcher for development

```
esy watch
```
